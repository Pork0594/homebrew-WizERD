# Homebrew Tap for WizERD

[![CI](https://github.com/Pork0594/homebrew-WizERD/actions/workflows/tests.yml/badge.svg)](https://github.com/Pork0594/homebrew-WizERD/actions) [![License](https://img.shields.io/pypi/l/wizerd.svg)](https://github.com/Pork0594/WizERD/blob/main/LICENSE)

[WizERD](https://github.com/Pork0594/WizERD) generates beautiful ER diagrams from PostgreSQL schema dumps.

## Requirements

- Python 3.9+
- Node.js 18+

## Installation

```bash
# Add the tap
brew tap Pork0594/WizERD

# Install wizerd
brew install wizerd
```

## Usage

For usage instructions, see the main [WizERD Repository](https://github.com/Pork0594/WizERD).

## Uninstalling

```bash
brew remove wizerd
brew untap Pork0594/WizERD
```
